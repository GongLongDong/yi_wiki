# 音素规则图谱

本页汇总当前较稳定的音素认知。它是工作图谱，不是封闭词典。每个音的“核心抽象”都应理解为能力池的压缩表达。

## 元音与变化类型

| 音 | 当前核心 | 源头/观察场景 | 主要维度 | 词例 |
|---|---|---|---|---|
| `a` | 开放、拉开、空间形成；全程开放变化 | 婴儿出生哭声，嘴唇从闭到开 | 空间形成、释放、量的展开、线性路径 | gap, cat, car |
| `o` | 闭合、整体、路径、循环、完整、安置 | 画圆；蛋的闭合空腔 | 首尾闭合、封闭空间、设定路径、循环、完整性 | top, box, room, rock |
| `u/w` | 闭合、接触、收缩、隐藏、状态切换 | 嘴从开到闭；状态切换 | 闭合、贴近、从显到隐或从闭到开 | put, cup, up, union |
| `e` | 目标点、结尾区、核心落点、结果 | 与 `a` 对比学习，尚未完全展开 | 变化的终点、目标、核心、落点 | get, rest |
| `i` | 单位化输出，可能带有流动或坍缩成一 | mem0/skill 稳定笔记，待展开 | 单位、细小输出、单点坍缩 | 待扩展 |

## 辅音与变化规则

| 音 | 当前核心 | 源头/观察场景 | 主要维度 | 词例 |
|---|---|---|---|---|
| `g` | 夹角控制、占据、获取、拥有 | 线对折成夹角；手形成夹角抓握 | 从无到有形成夹角，占据内部空间 | gap, get, bag, hug |
| `b` | 内部积累、膨胀、临界、裂开/爆炸 | 木柴/竹节受热后膨胀裂开或爆炸，`ba` 场景音 | 大、凸出、二分、边界、相反运动、爆发、反弹 | bulk, bi-, border, but, bomb, break |
| `p` | 面接触、放置、隔离、剥离空间 | put 放置动作；物体底部与表面贴合 | 大接触面、覆盖、安放、位置标记、空间分离 | put, place, point, park, cap, gap |
| `t` | 接触、支撑、进入并维持状态 | 下方支撑物体，上下参照形成 | 支撑、到达、接触、保持、方向性 | top, at, cat, put |
| `k/c` | 聚焦、锁定、攻克 | 吃生蛋 `ko` 场景；点接触敲蛋壳 | 小接触面、压强、硬碰硬、攻克、锁定 | cat, cut, key, lock, crack, focus |
| `s` | 细长连续、沿面滑动、扫描感知、扩散 | 蛇嘶声、吐舌、游走、细长身体 | 感知、滑移、细长、表面连续、扩散 | sense, smooth, slide, slim, sky |
| `r` | 指向、投射、抬升、目标关系 | 蛇抬身盯目标；种子破土 | 一点到目标、抬升、目的性、方向转换 | rise, car, rock, run |
| `h` | 相合、两端、两端之间 | 目前主要通过 `sh` 与两端振荡理解 | 两端关系、之间、合拢、压入 | shake, hug |
| `n` | 重复小接触输出、序列、痕迹、保留/更替 | mem0 稳定笔记，待独立展开 | 更替、保留、连续输出、范围收窄 | can, run, union, an |
| `y` | 环绕、整体包围、环绕式观察 | mem0 稳定笔记，待独立展开 | 环绕、扫视、周围、一体性 | sky, tyre |
| `l` | 对齐延伸、线性连续、结构传播 | skill 稳定笔记，待独立展开 | 线、轮廓、传播、连续 | 待扩展 |
| `v/f` | 差异显现/流动扩散 | skill 与 b/f 对比笔记，待展开 | 显现、分叉、扩散、持续释放 | 待扩展 |
| `m` | 外边界显现、覆盖、内部隐藏、溢出 | skill 稳定笔记，待展开 | 边界、覆盖、镜像、隐藏 | 待扩展 |

## 关键对比

### `a` vs `o`

| 维度 | `a` | `o` |
|---|---|---|
| 路径 | 开放线性 | 闭合环绕 |
| 方向 | 从 A 到 B | 绕一圈回到起点 |
| 结果 | 空间形成、展开 | 封闭空间、完整、循环 |
| 感觉 | 打开、释放 | 圈定、完成、安置 |

### `k/c` vs `p`

| 维度 | `k/c` | `p` |
|---|---|---|
| 接触方式 | 点、刃、尖 | 面 |
| 压强 | 大 | 小 |
| 典型结果 | 攻克、穿透、碎裂、锁定 | 覆盖、安放、隔离、标记位置 |
| 例子 | 敲蛋、刀切、钥匙攻克锁芯 | 放杯子、戴帽子、停车 |

### `b` vs `k/c`

| 维度 | `b` | `k/c` |
|---|---|---|
| 力方向 | 从内向外 | 向一点聚焦或锁定 |
| 变化倾向 | 膨胀、裂开、爆发 | 攻克、切入、锁定 |
| 典型结构 | 内部积累后突破边界 | 小接触面提高压强 |

## 组合结构索引

| 结构 | 抽象组合 | 典型词 |
|---|---|---|
| `ca` | 锁定性空间变化 | cat, cap, car |
| `co` | 锁定闭合空间，保护或核心 | core, code |
| `ap` | 空间变化经面接触剥离 | cap, gap |
| `op` | 安置/闭合后经面接触坍缩 | top |
| `to` | 支撑进入安置或朝向 | top, to |
| `ro` | 指向性旋转或滚动 | rock, roll |
| `sh` | 沿面 + 两端之间，形成振荡 | shake |
