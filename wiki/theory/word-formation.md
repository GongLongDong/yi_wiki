# 构词与坍缩

易音构词的核心是：音素按顺序组织成变化过程。词不是音素标签的相加，而是一条变化链。

## 基本角色

| 成分 | 角色 |
|---|---|
| 元音 | 变化类型，如开放、闭合、目标、状态切换 |
| 辅音 | 变化规则或动作模式，如支撑、聚焦、面接触、膨胀 |
| 半元音 | 混合角色，兼具变化流和规则性 |
| 尾辅音 | 可能收束或坍缩前序变化，但不是必然 |

## 音序原则

音序描述过程顺序。

例：`gap`

```text
g: 夹角占据空间
a: 开放空间变化
p: 面接触分离/剥离
=> 被占有的空间因分离而显出间隙
```

例：`top`

```text
t: 下方支撑
o: 安置/闭合空间
p: 面接触坍缩为稳定位置
=> 支撑并安置后形成顶
```

## 定义模式与应用模式

| 模式 | 说明 |
|---|---|
| 定义模式 | 词内音素组合描述完整变化过程 |
| 应用模式 | 词作为整体单元直接指代概念 |

`gap` 在定义模式中是一个产生间隙的变化过程；`a gap` 在应用模式中是一个已经坍缩的结果对象。

## 内部坍缩与外部坍缩

内部坍缩是词内变化走完后成为固有词义。

外部坍缩是词作为语法单位在句子中被名词化、完成体化或状态化。

两者要分开，否则会把词义结构和语法使用混在一起。

## 视角决定概念

同一场景可以产生多个概念。

抓苹果场景中：

| 观察焦点 | 概念 |
|---|---|
| 手指之间尚未闭合的空隙 | gap |
| 苹果进入夹角被获得 | get/grasp |
| 手闭合动作 | close/grip |

词不是客观场景的全部，而是观察者从场景中切出的一个变化视角。
