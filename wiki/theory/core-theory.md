# 核心理论

易音把语言看成一套从场景经验中生长出来的概念表达系统。它关注的不是“字母等于什么静态意义”，而是“声音在什么场景中与什么变化结构发生了绑定”。

## 1. 变化是基本单位

易音分析的第一对象不是词义，而是变化。

变化是在一个或多个维度上形成差异量的过程。只要至少一个维度发生差异，变化就已经出现。物体、状态、位置、名词意义，往往是变化过程完成后的结果。

例如“顶”不是先验存在的静态平面，而可以被分析为：

```text
下方支撑 -> 向上安置 -> 面接触稳定 -> 坍缩为顶
```

## 2. 场景先于抽象

一个音的意义不是凭空设计出来的，而来自可重复感知的场景。人在场景中观察到变化，经过记忆比对和认知积累，形成概念；场景中的声音再与这个概念发生强关联。

基本链条：

```text
场景变化 -> 观察/经历 -> 信息比对 -> 辨识差异 -> 概念形成 -> 声音成为载体
```

## 3. 声音优先，字形次之

当前易音的分析原则是声音优先。书写形式是视觉容器，声音才是概念变化结构的主要承载者。

这意味着：

- 不要把字母形状直接当成固定意义。
- 不要把现代拼写当成唯一证据。
- 同一字母在不同发音结构中，可能承担不同角色。

## 4. 音有能力池

一个音不是一个死定义，而是一组由源头场景释放出的表达能力。具体词只激活其中部分维度。

例如 `b` 可以从同一“内部积累后突破边界”的场景中产生多个维度：

- 膨胀、变大
- 二分、双
- 边界、边缘
- 相反运动
- 分离
- 爆发
- 反弹

词义越复杂，可能激活的维度越多。

## 5. 场景音与拆解音

重要修正：`ba` 这类声音可能首先是整体场景音，而不是 `b + a` 机械拼成的组合。`b` 和 `a` 是后来从整体场景音中拆解、稳定并复用出来的音素。

所以分析时要先判断：

1. 这个音是否可能是整体场景音？
2. 哪些部分是后续拆解出的稳定音素？
3. 这些音素在词中如何重新组合？
