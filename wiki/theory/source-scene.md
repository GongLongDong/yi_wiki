# 音源场景理论

音源场景是某个音最初与认知结构发生绑定的场景。它可以来自身体发音、自然声音、动物行为、工具操作、生存活动或重复出现的物理过程。

## 基本链条

```text
场景 -> 声音 -> 变化观察 -> 认知抽象 -> 概念载体
```

例如：

- 婴儿哭声中，嘴唇从闭到开，形成 `a` 的开放空间变化。
- 木柴或竹节受热膨胀，裂开或爆炸，伴随 `ba` 场景音，拆解出 `b` 的内部积累与突破。
- 吃生蛋时，敲击蛋壳发出 `ko` 声，力聚焦在点接触上，形成 `k/c` 的攻克与锁定。
- 蛇发出 `ssss`，游走、吐舌、细长身体形成 `s` 的感知、滑移、细长连续。

## 音源场景不是故事

音源场景的作用不是给词义编故事，而是提供可反复验证的变化结构。

一个好的音源场景应满足：

- 有可观察的自然或身体过程。
- 有清晰的开始、过程和结果。
- 能释放多个相关维度。
- 能解释多个词例，而不是只解释一个词。

## 场景音优先

有些声音最初可能是整体场景音。例如 `ba` 可能先是爆裂场景中的整体声音，然后才被拆解为 `b` 和 `a`。

因此，不能机械地从现代拼写倒推音素组合。应先判断整体音是否来自强场景，再分析拆解后的音素如何复用。

## 已知场景索引

| 音 | 场景 |
|---|---|
| `a` | 出生哭声与嘴唇从闭到开 |
| `b` | 木柴、竹节受热后内部积累、裂开或爆炸 |
| `g` | 线对折成夹角；手形成夹角抓握 |
| `k/c` | 吃生蛋时敲击蛋壳，点接触攻克硬壳 |
| `o` | 画圆与蛋的封闭空腔 |
| `p` | 物体放置时的面接触与空间标记 |
| `s` | 蛇的嘶声、吐舌、游走、细长身体 |
| `r` | 蛇抬身盯目标；种子破土 |
| `t` | 下方支撑物体并维持状态 |
