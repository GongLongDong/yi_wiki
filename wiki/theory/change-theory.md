# 变化理论

变化是在单个或多个维度上形成差异量的过程。

## 判断变化是否发生

设定开始状态和结束状态：

- 变化已发生：至少一个被观察维度上的差异量不为 0。
- 变化未发生：被观察的全部维度上差异量都为 0。

这里的“维度”不只包括三维空间，也包括时间、程度、数量、颜色、热量、显隐、边界、方向、接触方式等观察模式。

## 空间变化的基础地位

在当前易音体系中，空间性变化是最基础的变化类型。很多看似非空间的变化，都可以转回空间关系理解：

| 类型 | 空间化理解 |
|---|---|
| 时间 | 空间变化的转换，例如钟表指针运动 |
| 程度 | 刻度空间中的位置差 |
| 数量 | 单位对象或单位痕迹的空间分布 |
| 颜色 | 光波等微观空间运动 |
| 热量 | 微观粒子运动状态 |

## 绝对变化与相对变化

绝对变化考量所有维度，因此时间维度必然持续存在差异，绝对变化永远发生。

相对变化只聚焦某个或某几个维度。例如判断一个人是否移动，只看空间位移；判断一个物体是否变红，只看颜色维度。

易音分析通常处理相对变化：观察者选择某些维度，忽略其他维度，于是形成特定概念。

## 变化与坍缩

一个动态变化过程完成后，可以坍缩为结果状态。

例如：

```text
产生间隙的过程 -> 完成后 -> 一个 gap
支撑安置的过程 -> 完成后 -> top
切开过程 -> 完成后 -> cut/cut surface
```

这解释了为什么词义常常表现为静态对象，但其底层仍是动态过程。
