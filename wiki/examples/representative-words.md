# 代表性组合词例

## get

```text
g + e + t
获取/占据 + 目标落点 + 接触/停止
```

朝目标形成获取动作，变化落向目标，最终接触并完成获取。

## cat

```text
c/k + a + t
聚焦锁定 + 空间移动 + 接触并维持捕获状态
```

猫的命名可从捕猎行为理解：目光锁定目标，空间移动靠近，爪子接触并按压猎物。

## cut

```text
c/k + u + t
聚焦攻克 + 状态切换/打开 + 接触后维持分离
```

刀刃把力集中到极薄的线，攻克连续表面，使结构打开并维持分离。

## cap

```text
c/k + a + p
锁定目标空间 + 空间变化 + 面接触覆盖/隔离
```

帽子或盖子通过锁定、覆盖、贴合接触，把内外空间隔离开。

## cup

当前理解仍需继续确认，尤其是 `u` 的方向。

```text
c/k: 聚焦触发
u: 状态切换，从表面打开凹陷或从闭到开
p: 面性边界与稳定
```

## top

```text
t + o + p
支撑 + 安置/闭合空间关系 + 面接触坍缩
```

“顶”是支撑、安置、面接触稳定之后形成的关系结果。

## rock

```text
r + o + ck
指向性运动 + 旋转/闭合路径 + 硬点接触/锁定
```

可解释为硬质结构锁定，也可解释为指向性滚动、旋转和硬点碰撞后的摇动。

## shake

```text
sh + a + k
两端之间来回振荡 + 空间变化 + 端点反弹/锁定
```

`sh` 是沿面连续与两端之间相合关系的组合。

## b 系列

| 词例 | 激活维度 |
|---|---|
| bulk, big | 膨胀、变大 |
| bi-, bicycle | 裂开后从一到二 |
| border | 裂开产生边界 |
| but, battle, buy | 相反方向运动 |
| bye, butter | 分离 |
| bomb, burst | 爆发 |
| bump | 反弹 |

## s 系列

| 词例 | 激活维度 |
|---|---|
| sense | 蛇吐舌式感知 |
| smooth, slide | 沿表面连续滑动 |
| slim, silk | 细长连续 |
| sky | 视觉感知、聚焦、环绕扫视 |
