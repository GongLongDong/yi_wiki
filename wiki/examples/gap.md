# gap：夹角中的间隙

## 音素结构

```text
g + a + p
夹角占据空间 + 开放空间变化 + 面接触分离/剥离
```

## 场景

手伸出去抓一个苹果。

1. 手张开，苹果在对面。
2. 大拇指和其他手指逐渐向中间合拢。
3. 手形成夹角，这是 `g` 的占据空间。
4. 合拢过程发生开放式空间变化，这是 `a`。
5. 手指之间仍有距离，空间被分离出来，这是 `p`。
6. 如果继续闭合并抓住苹果，gap 消失。
7. 如果停在夹角形成但未闭合的中间态，就显出 gap。

## 核心洞察

`gap` 不是“一个静态空洞”的起点，而是抓握过程的中间态被观察者聚焦后坍缩成名词。

同一场景中：

| 观察焦点 | 概念 |
|---|---|
| 手指之间未填满的空隙 | gap |
| 苹果被获得 | get/grasp |
| 手闭合过程 | close/grip |

## 坍缩

| 形态 | 状态 |
|---|---|
| gap 动词 | 产生间隙的过程 |
| gapping | 间隙正在形成 |
| gapped | 过程完成后的结果状态 |
| a gap | 名词化结果 |
