# 音素总览

本页是音素入口。每个音素页面记录源头场景、变化结构、能力池、词例和待确认点。

|---|---|---|

## 待展开

见 [待展开音素](emerging.md)。

当前待重点研究：`u/w`、`e`、`n`、`y`、`l`、`m`、`f/v`、`i`、`d`、`z`、`j`、`th`、`ch`、`ng`。

## 重要对比

- `a` 与 `o`：开放线性 vs 闭合环绕。
- `k/c` 与 `p`：点/刃/尖接触 vs 面接触。
- `b` 与 `k/c`：内部向外突破 vs 力向一点聚焦。
- `s` 与 `r`：蛇的清音基础 vs 浊化后的指向抬升。
