# 音 b：内部积累、膨胀与裂开

## 源头场景

`b` 的核心场景是木柴、湿柴、竹节或类似封闭物在火中受热，内部水汽积累，最终裂开或爆炸，伴随 `ba` 爆裂声。

`ba` 很可能先是整体场景音，`b` 是后来从该场景音中拆解出的辅音规则。

## 核心变化链

```text
内部积累 -> 膨胀 -> 临界态 -> 边界突破 -> 裂开/爆炸/向外显现
```

临界态很重要。它既不是单纯膨胀，也不是已经裂开，而是性质即将转换的叠加状态。

## 能力池

| 观察窗口 | 概念 | 词例 |
|---|---|---|
| 膨胀 | 大、凸出、体积增加 | bulk, big, bulge |
| 裂开 | 断裂、分离 | break, bye |
| 数量 | 一分为二、双 | bi-, bicycle, binary |
| 边界 | 裂开产生分界线 | border, boundary |
| 相反运动 | 两半朝反方向弹开 | but, battle, buy |
| 爆炸 | 高能释放、不可复原扩散 | bomb, burst, blast |
| 反弹 | 碰撞后方向反转 | bump |
| 摆动 | 两侧围绕连接点摆动 | bee |

## 拓扑塑形

`b` 还可从“内部向外塑形”的角度解释形体形成：

- `bar`：从一端拉伸到另一端，形成条状物。
- `box`：从中心向外膨胀，形成边界和盒状空间。
- `bag`：膨胀形成边界，确定可占据空间。

## 方法提醒

不要把 `b` 简化成“膨胀”。膨胀只是其中一个观察窗口。完整结构是内部积累、临界转换、突破边界和向外显现。
