# 音 p：面接触、放置与空间剥离

## 源头动作

`p` 的核心动作是放置，典型词是 `put`。

```text
拿着物体 -> 向表面移动 -> 底部与表面贴合 -> 占据一块区域 -> 稳定放置
```

## 核心观察

`p` 强调两个表面贴合接触。接触后，这块区域从原本的大空间中被剥离、标记或隔离出来。

## 能力池

- 面接触
- 放置
- 覆盖
- 隔离
- 标记位置
- 从整体中剥离空间

## 词例

- `put`：放置。
- `place`：通过接触标记位置。
- `point`：指尖接触，从背景中标记出点。
- `park`：停车位或公园都是从大空间中划出的专属区域。
- `cap`：覆盖并隔离空间。
- `gap`：原本占有的部分分离后留下空隙。
- `top`：面接触稳定后坍缩为顶。

## 与 k/c 的对比

`k/c` 是小接触面、高压强、攻克；`p` 是大接触面、低压强、覆盖与稳定。
