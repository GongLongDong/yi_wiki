# 待展开音素

本页记录尚未写成定论的音素。它们可以作为后续学习和更新的入口。

| 音 | 当前线索 | 需要补充 |
|---|---|---|
| `u/w` | 闭合、接触、收缩、隐藏、状态切换；在 `put/cup/up/union` 中多次出现 | 明确开到闭、闭到开、显隐切换的条件 |
| `e` | 目标点、结尾区、核心落点、结果；与 `a` 互补 | 独立源头场景与词例链 |
| `n` | 重复小接触输出、序列、痕迹、保留、更替；影响 `an/can/run/union` | 源头场景与稳定能力池 |
| `y` | 环绕、周围、一体性、环绕式观察；见 `sky/tyre` | 源头场景与更多词例 |
| `l` | 对齐延伸、线、轮廓、结构传播 | 独立文档 |
| `m` | 外边界显现、覆盖、隐藏内部、镜像、溢出 | 独立文档 |
| `f/v` | `f` 偏持续流动扩散，`v` 偏差异显现与展开 | 区分发音和场景来源 |
| `i` | 单位化输出，可能结合流动与坍缩成一 | 词例验证 |
| `d/z/j/th/ch/ng` | 暂无稳定整理 | 后续逐个建立 |
