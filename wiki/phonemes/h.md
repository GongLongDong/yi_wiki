# 音 h：两端、相合与振荡

## 当前状态

`h` 的独立源头场景还需要继续展开。当前较稳定理解来自 `sh` 与 `shake`。

## 核心抽象

`h` 表达相合性变化规则。坍缩后可表达两端或两端之间的关系。

## sh 结构

```text
s: 沿表面连续移动
h: 两端之间/相合关系
sh: 两端之间的连续往返运动
```

## shake

```text
sh + a + k
两端之间来回振荡 + 空间变化 + 到端点时的反弹/锁定
```

## 待继续研究

需要为 `h` 建立更独立的源头场景，而不是只依赖 `sh` 组合。
