# 音 g：夹角占据与获取

## 源头结构

`g` 的核心来自线对折成夹角，以及手形成夹角抓握东西。

```text
直线 -> 弯曲/对折 -> 夹角形成 -> 夹角占据内部空间
```

人体层面：

```text
手张开 -> 手指向中间弯曲 -> 形成夹角 -> 包围/占据手心空间
```

## 核心抽象

夹角内部有没有东西不是关键。关键是“夹角从无到有形成，并占据了内部空间”。

因此 `g` 表达：

- 夹角控制
- 占据空间
- 包围
- 获取
- 拥有

## 词例

- `gap`：夹角占据空间，但尚未填满，于是显出间隙。
- `get`：夹角式获取朝目标落点完成。
- `bag`：边界膨胀形成空间后，由 `g` 确定占据。
- `hug`：两端包围、状态切换、夹角占据形成拥抱。

## 与方法论的关系

`g` 是“具象到抽象”的代表例：

```text
手抓东西 -> 夹角形成 -> 空间被占据 -> 获取/占有
```
