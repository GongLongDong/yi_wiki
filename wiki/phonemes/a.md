# 音 a：开放与空间形成

## 源头场景

`a` 的第一音源场景是婴儿出生时的哭声，以及发音中嘴唇从闭合到张开的过程。

```text
闭合 -> 气流冲出 -> 嘴唇逐渐张开 -> 空间形成 -> /a/
```

## 核心变化

最本质的变化是：原本没有空间，随着张开动作发生，嘴唇之间形成空间。

因此 `a` 表达：

- 开放
- 拉开
- 空间形成
- 从 A 到 B 的线性变化
- 变化的完整过程

## 与 o 的对比

| 维度 | a | o |
|---|---|---|
| 路径 | 线性开放 | 闭合环绕 |
| 方向 | 从起点到终点 | 绕回起点 |
| 状态 | 打开、释放 | 圈定、完整 |
| 结果 | 空间形成 | 封闭空间 |

## 词例

- `gap`：夹角中形成开放空间。
- `cat`：锁定目标后发生空间移动。
- `car`：被锁定/容纳的空间带着内容朝目标移动。

## 待继续观察

`a` 还可能表达量、程度、时间等维度上的展开，因为这些维度都可以转化为空间关系观察。
