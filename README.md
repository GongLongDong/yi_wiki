# 易音知识库

易音是一套从“音源场景、变化过程、认知抽象、声音载体”出发理解音义关系的语言分析项目。它不是把字母当成静态标签，而是尝试还原声音背后的场景经验和变化结构。

本仓库用于沉淀可持续更新的易音知识库。当前内容还不是最终体系，后续会随着新的讲解、修正和词例分析继续扩展。

## 当前稳定核心

- 变化是易音分析的基本单位。
- 音不是任意符号，而是与场景经验和认知抽象绑定的概念载体。
- 元音倾向表达变化类型，辅音倾向表达变化规则或动作模式。
- 一个音拥有表达能力池，具体词只激活其中部分维度。
- 词义常常是变化过程完成后的坍缩结果。
- 分析时先还原场景与变化，再进入现代词义。
