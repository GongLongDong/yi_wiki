# 例词分析库

本页不是完整词典，而是用于检验规则的样例库。每个例词都应能回到 [[03-phoneme-map|音素规则图谱]] ([音素规则图谱](03-phoneme-map.md)) 和 [[04-word-formation|构词与坍缩规则]] ([构词与坍缩规则](04-word-formation.md))。

## gap

```text
g + a + p
夹角占据空间 + 开放空间变化 + 面接触分离/剥离
```

源场景：手伸向苹果，手指形成夹角但尚未完全闭合。观察焦点不是“抓住苹果”，而是夹角中显出的空隙。

结论：`gap` 是抓握过程中的中间态被观察并坍缩为“间隙/缝隙”。

## get

```text
g + e + t
获取/占据 + 目标落点 + 接触/到达/停止
```

过程：朝目标形成获取动作，变化落向目标，最终接触并完成获取。与 `gap` 的区别在于观察焦点：`gap` 看尚未填满的空隙，`get` 看获取目标的完成。

## cat

```text
c/k + a + t
聚焦锁定 + 空间移动 + 接触并维持捕获状态
```

源场景：猫捕猎。目光和身体锁定目标，空间移动靠近，爪子接触并按压猎物。猫的命名取其捕猎行为特征，而不是动物静态形状。

## cut

```text
c/k + u + t
聚焦攻克 + 状态切换/打开 + 接触后维持分离状态
```

刀刃把力集中到极薄的线，攻克物体表面，使原本连续的结构被打开并维持分离。

## cap

```text
c/k + a + p
锁定目标空间 + 空间变化 + 面接触覆盖/隔离
```

帽子或盖子不是单纯“上方物”，而是通过锁定空间、覆盖、贴合接触，把内外空间隔离开。

## cup

当前较稳定理解：`cup` 不是简单的“收拢容器”，而更接近“聚焦触发，从表面挖出/形成凹陷空间，再通过面性边界稳定出来”。

```text
c/k: 聚焦触发
u: 状态切换，从闭到开或从表面打开凹陷
p: 面性边界/放置性稳定
```

仍需继续确认 `u` 在该词中的方向性。

## top

```text
t + o + p
支撑 + 安置/闭合空间关系 + 面接触坍缩
```

“顶”不是静态上表面，而是从下方支撑、上方被安置、最后在接触面上稳定的关系坍缩。

## put

```text
p + u + t
面接触放置 + 状态切换/贴近 + 支撑维持
```

放置动作的核心是物体与表面贴合，在接触处标记出一个位置，并由支撑关系维持稳定状态。

## rock

```text
r + o + ck
指向性运动 + 旋转/闭合路径 + 硬点接触/锁定
```

可解释两个相关方向：作为石头，是硬质结构锁定；作为摇动/滚动，是指向性运动与旋转路径、硬点碰撞的组合。

## shake

```text
sh + a + k
两端之间的连续振荡 + 空间变化 + 端点反弹/锁定
```

`sh` 来自 `s + h`：沿面连续移动与两端之间的相合关系，形成两端之间来回运动。`k` 表示到达端点时那一下反弹或锁定。

## b 系列

`b` 的词例应按观察窗口整理。

| 观察窗口 | 词例 | 核心说明 |
|---|---|---|
| 膨胀/变大 | bulk, big | 内部积累向外扩张 |
| 二分 | bi-, bicycle | 裂开后从一到二 |
| 边界 | border, boundary | 裂开产生分界线 |
| 相反运动 | but, battle, buy | 两半或两方朝相反方向运动 |
| 分离 | bye, butter | 原整体中分出一部分 |
| 爆发 | bomb, burst, blast | 内部积累突破为高能释放 |
| 反弹 | bump | 触碰后方向反转 |

## s 系列

| 观察窗口 | 词例 | 核心说明 |
|---|---|---|
| 感知/探测 | sense | 蛇吐舌扫描环境 |
| 沿表面滑动 | smooth, slide | 贴着表面连续流畅移动 |
| 细长形态 | slim, silk | 蛇身/丝线式的连续细长 |
| 视觉扫视 | sky | 视觉感知 + 聚焦 + 环绕扫视 |

## 待扩展词例

- `an`：`a + n`，开放空间被重复更替/保留，逐渐收窄为确定区域。
- `union`：`u + n` 激活向心聚拢与交替保留。
- `tyre`：支撑、环绕、指向关系的组合。
- `core`：锁定闭合空间与功能指向的组合。
