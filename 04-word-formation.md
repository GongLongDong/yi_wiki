# 构词与坍缩规则

易音的构词不是“字母等于词义”的表格，而是音素按顺序描述一个变化过程。一个词可以在定义模式下展开为过程，也可以在应用模式下作为整体概念使用。

## 1. 基本角色

| 成分 | 角色 | 说明 |
|---|---|---|
| 元音 | 变化类型 | 表达路径、开合、闭合、目标、状态切换等变化类型 |
| 辅音 | 变化规则/动作模式 | 表达接触方式、力的方向、支撑、聚焦、扩散、占据等规则 |
| 半元音 | 混合角色 | 兼具变化流与规则性，需按具体结构判断 |
| 尾辅音 | 可能坍缩前序变化 | 但不自动坍缩，需看结构、发音、语义倾向 |

## 2. 音序原则

音的排列顺序描述过程顺序。先出现的音通常给出先发生的变化或规则，后出现的音给出后续变化、接触、结果或坍缩。

例：`gap`

```text
g: 形成夹角，占据空间
a: 开放式空间变化
p: 面接触分离/剥离空间
=> 被占有的空间因分离而显出间隙
```

例：`top`

```text
t: 从下方支撑
o: 安置/闭合空间关系
p: 面接触坍缩为稳定位置
=> 支撑并安置后形成“顶/上表面”
```

## 3. 定义模式与应用模式

| 模式 | 含义 | 例子 |
|---|---|---|
| 定义模式 | 词内音素组合描述完整变化过程 | `gap` 展开为形成夹角、空间变化、分离 |
| 应用模式 | 词作为整体单元直接指代概念 | `a gap` 指“一个间隙” |

定义模式强调过程，应用模式强调使用。

## 4. 坍缩规则

坍缩的本质是“变化完成后，过程被压缩为结果状态”。

| 形态 | 状态 | 是否坍缩 |
|---|---|---|
| gap 动词原形 | 可描述产生间隙的过程 | 未完全坍缩 |
| gapping | 进行中 | 未坍缩 |
| gapped | 已完成 | 坍缩为结果 |
| a gap | 名词化 | 坍缩为对象/结果 |

要区分两种坍缩：

- 内部坍缩：词内音素组合描述的变化已经在词义中走完。
- 外部坍缩：词作为语法单位被句子使用，按词性、时态、语境坍缩。

## 5. 抽象先行，具象后入

分析一个词时，应先组合抽象音义，再根据词义需要引入具象场景。

错误路径：

```text
先看到现代词义 -> 找几个音硬凑解释
```

推荐路径：

```text
音素能力池 -> 音序过程 -> 抽象骨架 -> 词义所需维度 -> 具象化解释 -> 自证
```

## 6. 视角决定概念

同一场景可以产生多个词，因为观察焦点不同。

抓苹果场景：

| 观察焦点 | 概念 |
|---|---|
| 大拇指与手指之间的空隙 | gap |
| 苹果被手包围并获得 | get/grasp |
| 手闭合动作 | close/grip |

因此，一个词不是客观场景的全部，而是某个观察焦点切出的变化结构。
