# 音h — 相合性变化

## 基本信息

| 属性 | 内容 |
|------|------|
| 音 | /h/ |
| 核心抽象 | **相合性变化规则** |
| 坍缩后 | 表达**两端**或**两端之间** |

## 在sh中的应用

sh = s（沿表面连续移动）+ h（两端之间的相合性变化）

→ **从一端到另一端之间的连续运动**，即在两端/两极之间的振荡/弹跳运动

## sh在shake中

**shake的完整过程：** 乒乓球在桌面和空中之间弹跳
- sh = 两端之间来回振荡（桌面↔空中）
- a = 空间变化
- k（/k/）= 到达端点时的那一下锁定/反弹

## 基本运动模式汇总

| 模式 | 音 | 描述 |
|-----|---|------|
| 直线 | a | 从A到B，单向 |
| 圆周 | o | 绕一圈回来，闭合循环 |
| 点接触 | k(c) | 力聚焦于一点，攻克/锁定 |
| 面接触 | p | 大面积贴合，隔离/覆盖 |
| 支撑 | t | 从下方抵住 |
| 膨胀/发散 | b | 从内向外扩张 |
| 指向性 | r | 朝特定目标方向运动 |
| 沿表面 | s | 贴着表面连续移动 |
| 相合/两端 | h | 两端之间，相合性变化 |
| 振荡 | sh | 两端之间来回运动 |
