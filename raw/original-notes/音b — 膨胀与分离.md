# 音b — 膨胀与分离

## 音源场景

篝火中竹子/湿柴受热，内部水汽膨胀，最终裂开或爆炸，伴随 **ba** 爆裂声。

场景高频，上古每日生火必遇。

同一场景中，具体条件决定路线：有缝的裂（干粗木、竹节有缝），封闭的爆（完整竹节、河卵石）。

## 从 ba 到 b

ba 是场景中直接触发的自然声音（整体音），b 是从 ba 中分解提取的抽象辅音音素。

```
自然声 ba → 分解提取 → 音b → 体系化规则组合 → 构词
```

此路径为印欧语言构词逻辑：分解为辅音/元音/半元音，通过体系化规则组合构建单词。规则体系逐代继承和发展。

## 变化规则

b 的变化规则：**内部积累 → 突破边界 → 向外显现**

### 变化链结构

```
变化A（膨胀）──量积累──► 质变点/临界态（叠加态）──退相干──► 变化B（裂开）
                                                      │
                                                      └──► 变化C（爆炸）
```

> [!important] 关键要点
> - 质变点 = 临界态 = 叠加态，**是膨胀的质变点**
> - 临界态既非膨胀也非裂开/爆炸，性质模糊，短暂
> - 退相干（Decoherence）：叠加态塌缩为确定结果
> - 内因决定路线：温和渗透/静态力 → 裂开；持续加码/封闭失控 → 爆炸
> - A → B/C 是**性质转换**，非程度增减
> - 整体变化（A→B / A→C）可定义为一个整体变化，也可分段观察

## 四个阶段及关联概念

### 阶段1：膨胀（未裂开/爆炸前）

核心特征：体积向外扩大，边界向外延申。

**觉察方式**：通过边缘是否向外延申。二元比对 —— t0 边缘位置 ⟷ tn 边缘位置，位移量 > 0 且方向向外 → 膨胀。

> [!tip] 人不直接感知体积，而是通过观察边缘的位移量和方向来间接判定膨胀。

#### 几何模式

**全局膨胀** — 球体 V1 → V2，所有边缘同步向外扩张

二元：中心到边的距离 R0 ⟷ Rn，Rn > R0 → 向外

- **bulk** — 体积大/大量
- **big** — 大

**局部凸出** — 某点突破表面向外凸起，其余不变

二元：不动点位置 ⟷ 凸出点位置，位移背离原表面法线 → 凸出

- **bold** — 突出/醒目（物理世界）、大胆/勇敢（意识世界，物理结构投射）
- **bulge** — 鼓起/凸出

**多点凸出** — 表面多处同时局部凸起

- **bunch** — 成簇/成团

### 阶段2：临界态（质变点）

核心特征：膨胀推到极限，边界即将失效。**叠加态** —— 既在膨胀又在裂开，两种状态并存。

ba 声恰在此时触发 —— 与裂开瞬间**并行**发生，关联最强。

### 阶段3：裂开（退相干结果A）

核心特征：边界突破，一分为二或多块，各部分仍保留。

| 维度 | 概念 | 例词 |
|------|------|------|
| 数量 | 二/双 | bi-, bicycle, binary, bilateral |
| 边界线 | 边界/边缘 | border, boundary, brink |
| 空间关系 | 相邻/旁边 | by |
| 运动方向 | 相反/对抗 | but, battle |
| 断裂本身 | 断裂 | break |
| 反向运动 | 反弹 | bump |
| 摆动 | 摆动 | bee |

### 阶段4：爆炸（退相干结果C）

核心特征：结构摧毁，能量瞬间向外释放扩散，不可复原。

| 维度 | 概念 | 例词 |
|------|------|------|
| 爆发 | 爆发 | bomb, burst, blast |
| 体积 | 大/大量 | bulk |

## 几何物理演示

| 阶段 | 演示方式 |
|------|---------|
| 全局膨胀 | 球体 V1 → V2，所有边缘同步向外扩张 |
| 局部凸出 | 球体表面某一点突破向外，参照周围未动点 |
| 多点凸出 | 球体表面多处同时起泡 |
| 临界态 | 球体边缘拉到透明，即将破裂的瞬间 |
| 裂开 | 球体分为两半，各自运动方向相反 |
| 爆炸 | 球体结构摧毁，碎片向外飞散 |

## 拓扑塑形思维

古人从陶器/泥器制作中衍生出"用b表达形状形成"的思维：

- **bar** — 从一端进行拓扑拉伸至另一端 → 条状物
- **box** — 从中心向外膨胀的塑形模式 → 方形盒子
- **bag** — 膨胀形成边界，确定出空间 → 袋子

## b vs f 对比

> [!info]

| 维度 | b | f |
|---|---|---|
| 音源 | ba 爆裂声 | fu 吹气声 |
| 节奏 | 积聚 → 瞬间释放 | 持续释放 → 扩散 |
| 能量 | 高、瞬间 | 均匀、持久 |
| 概念偏向 | 爆、裂、凸、突、大 | 流、散、平、养 |

f 同样是向外，区别在于节奏 —— b 爆发性，f 流动性。详见 [[音f — 流动与扩散]]。

## 概念形成

从上述多个维度的观察和认知，坍缩为音b的核心概念：**从内部向外部的扩张、展开、分离**。

理解b的关键是进入那个**木柴受热 → 内部膨胀 → 裂开/爆炸的连续场景**中观察变化，而不是记"b=膨胀"。

> [!summary] 不同观察窗口产出不同概念
> - 观察膨胀 → bulk、bold、big
> - 观察临界态 → ba声关联本身
> - 观察裂开 → bi-、border、by、but、break、bump
> - 观察爆炸 → bomb、burst
> - 观察膨胀→裂开 全过程 → 整体视角，概念综合
