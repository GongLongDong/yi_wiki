# 易音 — 完整学习报告

> [!abstract] 项目核心
> 基于场景体验的语言分析理论：**场景 → 认知 → 概念 → 音作为概念载体**
>
> 语言不是任意的符号系统，音与义之间的关联根植于人类在场景中的**身体体验**和**认知抽象**。

---

## 一、理论基础

### 1. 变化理论

变化是在单个或多个维度上形成差异量的**过程**。

**维度体系：**
- 空间三维（长宽高）— 基础维度
- 时间 — 空间变化的转换（钟表指针移动→时间）
- 程度/量 — 通过刻度空间位置体现
- 颜色 — 光波微观空间运动

**两种变化类型：**
- **绝对性变化** — 考量所有维度，时间维度必然存在差异量 → 永远存在
- **相对性变化** — 聚焦某个或某几个维度，忽略其他

### 2. 认知方法论

**从混沌到概念的完整链条：**
```
混沌状态（一切信息未分化）
    ↓
需求驱动（生存需要产生关注方向）
    ↓
选择性观察（从混沌中选取关注信息）
    ↓
获取信息（各维度变化信息）
    ↓
信息比对（同一维度前后变化产生信息差）
    ↓
辨识差异（从信息差中识别出有意义的变化）
    ↓
认知积累（多次辨识的认知沉淀）
    ↓
概念形成（对一类变化模式的抽象提炼）
    ↓
寻找载体（需要一个音/形来表达这个概念）
    ↓
表达与交流（通过载体传递概念）
```

**分化理论：** 不是A和B两个独立东西拼到一起，而是从整体内部"长出"边界（纸张折叠模型）。

**共识理论：** 概念是私人的——每个人的认知体系都不同。沟通的有效性取决于**交集的大小**。共识不是"完全一致"，而是"交集足够大，使得沟通在目标范围内有效"。

**语言代沟（信息衰减）：** 每一代语言传递都会丢失一层"为什么"——原初场景被遗忘，只有符号被保留下来。就像画老虎：起初画师观察真实老虎后画出简笔虎，一代代传下去越画越简化，后来的人只知其形不知其源头。

### 3. 构词规则

| 成分 | 角色 |
|------|------|
| **元音** | 表达**变化类型**（如 a = 空间性变化，o = 圆形/闭合变化） |
| **辅音** | 表达**变化规则/力作用方式**（如 k = 聚焦锁定，p = 面接触分离） |
| **辅音+元音** | 将辅音的**变化规则**赋予元音的**变化类型** |

**单词的两种模式：**

| 模式 | 说明 | 例 |
|------|------|-----|
| **定义模式** | 音素组合描述一个完整的变化过程 | gap = g(占有)+a(空间)+p(分离) |
| **应用模式** | 直接使用单词指代概念 | gap = "间隙"（整体单元） |

**变化坍缩：** 一个动态变化过程，当它"完成"时，就从过程坍缩为结果状态。物或状态只是变化坍缩的结果。

**音序组合的意义：** 音的排列顺序**决定了描述过程的先后顺序**。先发生的变化在前，后发生的变化在后。

## 二、完整音素体系

### 元音——变化类型

| 音 | 运动模式 | 源头场景 | 过程描述 | 坍缩后 |
|---|---------|---------|---------|--------|
| **a** | 直线/开放 | 婴儿出生哭声（嘴从闭到开） | 从A到B，单向，有起点终点 | 空间 |
| **o** | 圆周/闭合 | 画圆/蛋的内部空腔 | 绕一圈回到起点，首尾相接 | 圆形/封闭空间 |
| **/u/** | 收缩/闭合 | 嘴从开到闭的压缩过程 | 空间在缩小→趋于封闭 | 封闭/贴合 |
| **/ʌ/** | 爆裂/从闭到开 | a的变体 | 从闭合到开的瞬间爆发 | 切换/打开 |
| **e** | 目标/结尾 | （与a对比讲解） | 变化的目标指向 | 目标/终点 |

**a / o / u / ʌ 四者关系（源于嘴巴从闭到开的连续过程）：**
```
嘴唇完全闭合 → /u/(闭合收缩) → /ʌ/(从闭到开的瞬间爆发) → /a/(张开完成，持续开放)
```

| 音 | 嘴的状态 | 变化方向 |
|---|---------|---------|
| /u/ | 从开到闭 | 空间在缩小→趋于封闭 |
| /ʌ/ | 从闭到开的瞬间 | 临界爆发，从封闭到打开的切换 |
| /a/ | 张开完成 | 空间形成且持续开放 |

### 辅音——变化规则

#### b — 膨胀/发散/分离

| 属性 | 内容 |
|------|------|
| 源头场景 | 木柴受热膨胀→裂开/爆炸 |
| 核心抽象 | **从内部向外部的扩张运动** |
| 关键词 | **驱动力** |

多维度分支：

| 观察焦点 | 认知形成 | 例词 |
|---------|---------|------|
| 数量 | 双/二 | bicycle |
| 空间关系 | 相邻/旁边 | by |
| 运动方向 | 相反/对抗 | but, battle |
| 分离 | 分离/告别 | bye |
| 边界 | 边界/边缘 | border, box |
| 膨胀过程 | 膨胀/变大 | bulk |
| 能量/速度 | 爆发性/爆炸 | bomb, burst |
| 反弹 | 反弹/回弹 | bump |
| 摆动 | 角度摆动 | bee |

拓扑塑形（来自陶器制作思维）：

| 塑形模式 | 例词 |
|---------|------|
| 端到端拉伸 | bar（条状物） |
| 中心向外膨胀 | box（边界） |
| 膨胀形成封闭空间 | bag |

#### p — 面接触分离

| 属性 | 内容 |
|------|------|
| 核心动作 | put（放置） |
| 核心抽象 | **两个表面贴合接触，从整体中隔离/剥离出空间区域** |
| 与k对比 | p=大接触面→覆盖/隔离，k=小接触面→攻克/穿透 |

例词：put, place, point, park, cap, gap, top, cup, up

#### t — 支撑 → 维持状态

| 属性 | 内容 |
|------|------|
| 核心抽象 | **从下方支撑 → 进入并保持某个状态** |
| 本质 | 支撑是手段，**维持状态**是目的 |

t的抽象链：
```
支撑（物理）→ 维持状态（功能）→ 方向性/朝向（延伸）
```

#### k(c) — 聚焦性锁定/攻克

| 属性 | 内容 |
|------|------|
| 源头场景（ko） | 吃生蛋：蛋是球体，接触面时一个点接触，全部重量聚焦到那个点 |
| 核心抽象 | **将力约束在较小的接触面上（点/刃/尖），通过增大压强实现攻克或锁定** |
| 与p对比 | 同样压力下，k=小接触面→攻克，p=大接触面→覆盖 |
| 统一模式 | **聚焦→锁定→攻克**（递进统一） |
| 硬质性分支 | 硬表面+聚焦力→碎裂（蛋壳、石头） |
| 保护性分支 | co=锁定空间→防御/保护（core） |

例词：cat, cut, key, lock, can, cap, focus, crack, rock

#### s — 沿表面滑移/感知/细长

| 属性 | 内容 |
|------|------|
| 源头场景 | 蛇（自然拟声："ssss"嘶声） |

多维度分支：

| 观察焦点 | 认知形成 | 例词 |
|---------|---------|------|
| 蛇吐舌探测环境 | **感知/感觉** | sense |
| 蛇游走沿表面滑动 | **沿表面连续移动** | smooth, slide |
| 蛇身躯细长形态 | **细长/纤细** | slim |
| 面的铺展 | **扩散/延伸性** | bus, spread, space |
| 视觉观察 | **视觉感知** | sky |

#### r — 指向性变化

| 属性 | 内容 |
|------|------|
| 源头场景 | 蛇遇侵犯者，抬起身子，盯住目标 |
| 核心抽象 | **指向性变化规则**——从一点向目标投射 |
| 与k对比 | k=四周向一点聚焦，r=从一点向目标投射 |
| 与s的关系 | s的清音基础，r为s的浊化，但具有独立场景 |

延伸含义：抬升（rise）、指向目的性（car）、方向切换（碰撞后改变方向）

#### h — 相合性变化

| 属性 | 内容 |
|------|------|
| 核心抽象 | **相合性变化规则**——坍缩后表达两端或两端之间 |

**sh** = s（沿表面）+ h（两端之间）= 从一端到另一端之间的连续运动，即振荡/弹跳

例：shake = sh(两端振荡) + a(空间) + k(端点反弹锁定)

#### n — 更替/保留

| 属性 | 内容 |
|------|------|
| 核心抽象 | 更替/保留——运动中的交替过程，坍缩后的保留状态 |
| 作用 | 连接交替中的不同部分/状态 |
| 例词 | can(容器: 锁住空间+更替内容), run(爆发+切换+交替保留), union(聚拢+交替+?) |

#### g — 获取/占有

| 属性 | 内容 |
|------|------|
| 源头场景 | 线对折形成夹角 |
| 核心抽象 | **夹角占据空间** |
| 应用 | 张口吃（嘴上下形成夹角占据食物空间）→ 获取/占有 |

例词：gap(夹角分离→缺口), get(夹角获取+目标+停止), bag(膨胀边界+空间+夹角), hug(两端+切换+夹角)

#### y — 环绕

| 属性 | 内容 |
|------|------|
| 核心抽象 | **环绕运动/环绕式观察** |
| 例词 | sky(视觉感知+聚焦+环绕扫视) |

### b与c的互逆关系

| 维度 | b（膨胀/发散） | c/k（锁定/收敛） |
|------|--------------|-----------------|
| 力的方向 | 从内向外推 | 从外向内收/触发 |
| 强化u方向 | bu=从闭到开（反向切换） | cu=从闭到开（聚焦触发形变） |
| 例证 | but(转折=b推动朝对立面切换) | cut(切割)、cup(挖坑) |
| 典型运动 | 爆炸、膨胀、裂开 | 锁定、攻克、切入 |

## 五、核心方法论

### 分析方法：抽象层 → 具象层

**第一步：抽象层组合**
列出每个音的抽象含义，不做具象化联想。仅从抽象面看它们组合后可能表达的核心概念框架。

**第二步：看词义**
根据该词的已知语义，确定分析方向。

**第三步：合理引入具象**
根据词义需求，从音初始场景所能表达的范畴中，选取合理的具象层含义。
- 合理性依据：具象含义必须能从该音的源头场景中自然推导出来
- 方向性由词义决定，而非音本身固定（如u的状态切换方向根据词义可开可闭）

**第四步：验证**
整个分析是否自洽？抽象→具象的链条是否合理？是否有杜撰的成分？

### 关键原则

| 原则 | 说明 |
|------|------|
| **动态过程优先** | 物/状态只是变化坍缩的结果，先还原完整变化过程 |
| **ba是场景音** | 不是b+a的组合，是整体被感知后拆解的 |
| **音不是设计工具** | 音是场景的一部分，不是在外部"设计"来表达概念的 |
| **多维度分析** | 一个音可以从多个维度同时解读，不锁死一个维度 |
| **不以现代语义束缚** | 不被现代词义束缚，回溯到物理过程 |
| **音具备多种表达能力** | 特征是潜力池，提取由词义需要决定，构词者可能只引入部分 |
| **先抽象层组合** | 先抽象层组合，再据语义引入具象层 |
| **自证** | 通过自然现象验证而非每次要老师确认 |

### 常见错误

| 错误 | 例子 | 正确 |
|------|------|------|
| 用坍缩结果代替过程 | "o=圆" | "o=沿路径环绕一圈回到起点的运动过程" |
| 一个音包揽整个词义 | but的转折只归因于b | 转折是b+ʌ+t共同完成 |
| 否定音的作用 | "转折不该归因于b" | b不是转折本身，但b是驱动转折发生的力 |
| 未学就推测 | 没学过h就解释sh | 先学h再理解sh |
| 杜撰场景 | 石头滚动与but无关 | 不从自然现象出发就是杜撰 |

## 八、方法论修正记录（最重要）

1. **"ba是场景音，不是你所说的组合音"** — 音不是拼凑出来的，是整体场景中被感知后拆解的
2. **"物或状态只是变化坍缩的结果"** — 必须从动态过程理解，不能从静态结果出发
3. **"你不需要我确认"** — 学会通过自然现象自证，减少依赖
4. **"不要锁死一个维度，多维度"** — 同一音在同一词中可同时从多个维度解读
5. **"根据其想要表达语义决定，而非固定"** — u的方向由词义决定
6. **"先采用音的抽象性含义"** — 先抽象层组合，再据语义引入具象层
7. **"音具备许多表达能力，但作为构词者，可能只引入部分"** — 特征是潜力池，提取由词义需要决定
8. **"找到两个参照点，比对空间关系"** — 概念形成于参照系的空间关系对比
