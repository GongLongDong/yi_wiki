# 音t — 支撑与接触

## 变化过程

t描述的是**支撑**——从下方提供支撑力，支撑点与上方物体贴合接触。

### 支撑的变化过程

1. 起始：一个待放置的物体，一个下方支撑面
2. 物体向下移动，接近支撑面
3. **物体与支撑面接触**——两者贴合
4. 支撑面从下方托住物体，提供稳定的支撑力
5. 结束：物体被稳定支撑住，双方处于接触状态

### 关键观察：空间关系对比

支撑点在下方，被支撑物在上方——通过这种**空间关系的对比**，才形成了"上/下"的概念。

> [!quote] 用户原话
> "支撑点和安置物两者的空间关系对比形成的概念"

如果只有一个孤立的东西存在于空间中，是不会有"上"或"下"的概念的。"上"之所以为"上"，是因为存在下方一个支撑点作为参照系。

## 从支撑到方向性

t的自然延伸逻辑：
- 支撑点在**下** → 被支撑物在**上**
- 从下指向上 → **方向性（朝向）**
- 指向某个目标 → 运动向目标**靠近/到达**

## t在单词中的体现

**top**
- t(支撑)+o(安置)+p(面接触放置)
- 完整过程：从下方支撑 → 向上抬升安置 → 接触面坍缩为顶
- "顶"不是天然在那的——是运动抬升到那个位置、由支撑托住、在接触面处固定后，才有了"顶"

**at**
- a(变化) + t(朝向/接触)
- 过程：变化朝向某个方向/状态 → 最终到达/接触 → 按压住
- t从"朝向"自然演化为"到达"→"接触"→"按压"
- at是一个音序结构单元

**cat**
- c(聚焦锁定)+a(空间变化)+t(支撑接触)
- 猫捕猎：聚焦锁定(c)→空间移动靠近(a)→爪子接触并支撑按压住猎物(t)

**put**
- p(面接触放置)+(u待学)+t(支撑)
- 放置后支撑稳定的状态
