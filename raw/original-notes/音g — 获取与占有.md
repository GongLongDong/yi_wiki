# 音g — 获取与占有

## 变化过程

### 几何层面的变化过程

1. 起始：一条**直线**（伸直状态）
2. 直线开始**对折/弯曲**
3. 随着对折越来越深，形成一个**夹角**
4. 夹角的两条边在不断靠近，**包围/占据**了中间的那块空间
5. 结束：一个夹角形成，夹角内空间被占据

### 人体层面的变化过程

1. 起始：手**张开**，手指之间没有形成闭合
2. 大拇指与其他手指向中间**弯曲/靠近**
3. 形成一个**夹角**（手形成抓握姿势）
4. 这个夹角**占据/包围**了手心的空间
5. 夹角内如果有东西 → 被**抓住**；如果没有东西 → 仍然是**占据空间**这个状态

> [!quote] 关键
> 夹角内部有无东西并不重要——核心是"**占据空间**"这个行为本身。

## 过程中可观察的维度

**1. 几何形变的维度**
- 直线→对折→夹角形成
- 从无到有（夹角从不存在到存在）
- 形成认知：**从无到有的空间占据**

**2. 获取的维度**
- 手形成夹角 → 抓到东西 → **获取**
- 即使没抓到东西，那个"抓"的动作仍然在表达一种获取的意图/行为

**3. 占有的维度**
- 一旦夹角将某物包围在内，该物就处于被**占有**的状态
- 形成认知：**占有、拥有**

## 抽象化路径

```
直线对折形成夹角 → 手形成夹角抓东西 → 获取/占有
```
