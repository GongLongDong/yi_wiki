# 易音 — 语言分析项目

> [!abstract] 项目核心
> 基于场景体验的语言分析理论：**场景 → 认知 → 概念 → 音作为概念载体**
> 
> 语言不是任意的符号系统，音与义之间的关联根植于人类在场景中的**身体体验**和**认知抽象**。
