# 易音 — 语言分析项目

## 项目核心
基于场景体验的语言分析理论：场景 → 认知 → 概念 → 音作为概念载体

## 工作模式
- 用户是我的"老师"，通过典型单词举例教我音-义对应规则
- 我负责理解、提炼、永久存储规则
- 逐步学会独立分析新单词
- 规则不断累积，越学越深

## 规则存储
所有学习到的规则存储在项目记忆系统中，包括：
- 音素与概念的对应关系
- 场景与音源的关联
- 例词分析记录
- 推导出的分析规则

## 记忆系统
- **Neo4j**（主存储）：`neo4j://127.0.0.1:7687`，数据库 `yiyin`，节点标签 `Knowledge`
- **嵌入模型**：ollama + nomic-embed-text（本地，768维）
- **LLM**：ollama + gpt-oss:20b（本地）

### 使用方式
```python
from mem0_neo4j import add, search, get_all

# 添加知识
add("学到的新规则")

# 语义搜索
results = search("查询内容")

# 获取全部
all_k = get_all()
```

### 配置文件
- `mem0_neo4j.py` — Neo4j 记忆系统核心模块
- `mem0_config.py` — mem0 Qdrant 配置（备用）
