# 构词规则

## 核心：规则 × 变化

> [!important] 基本原则
> - **元音** = 表达**变化类型**（如 [[音a — 空间变化\|a]] = 空间变化）
> - **辅音** = 表达**变化规则**（如 [[音g — 获取与占有\|g]] = 形成夹角占据空间的规则）
> - **辅音 + 元音** = 将辅音的**变化规则**赋予元音的**变化类型**

## 核心原则：以动态过程理解单词

> [!important] 物或状态只是变化坍缩的结果
> 
> 理解一个词的关键是**先描述它背后的变化过程**——起点是什么，经历了什么，每一步发生了什么变化——而不是先记住它坍缩后的静态结论。
> 
> 例：top不是"上面那个平面"，而是"从下往上抬升→支撑托住→面接触固定"的完整过程。只是这个过程走完后坍缩为"顶"这个结果。

## 概念形成于空间关系对比

> "支撑点和安置物两者的空间关系对比形成的概念"

一个概念之所以成立，往往不是因为事物本身的固有属性，而是**它与参照物之间的空间关系对比**。没有"下"就没有"上"，没有"散"就没有"聚"。

### 示例：ga

| 成分 | 含义 |
|------|------|
| g（规则） | 形成夹角占据空间（从无到有） |
| a（变化） | 空间变化（空间从无到形成） |
| **ga（组合）** | 将"形成夹角占据空间"规则赋予"空间变化" |

## 单词的两种模式

### 1. 定义模式

> 通过完整的音素组合过程来描述概念。

- 每个音素贡献语义，组合起来讲述一个完整的变化过程
- 概念本身始终是**动态过程**
- 例：`gap` = g(占有) + a(空间变化) + p(面接触分离/剥离)

### 2. 应用模式

> 在表达中直接使用单词指代其概念。

- 不拆解音素，将单词作为一个整体单元使用
- 例：`gap` = "间隙、缝隙"

## 变化坍缩

> [!info] 坍缩的本质：**完成**
> 
> 一个动态变化过程，当它"完成"时，就从过程坍缩为结果状态。

### 坍缩条件

| 应用形态 | 是否完成 | 是否坍缩 |
|---------|---------|---------|
| gap（动词原形） | 尚未完成 | 否 → 过程态 |
| gapping（进行体） | 进行中 | 否 → 过程态 |
| gapped（完成体） | 已完成 | **是 → 结果态** |
| a gap（名词） | 隐含已完成 | **是 → 结果态** |

### 规则总结

- **定义模式下**：概念始终是动态过程
- **应用模式下**坍缩与否取决于：
  1. **词性**：动词（过程态）vs 名词/状态词（结果态）
  2. **时态**：进行体/未完成体（过程态）vs 完成体（结果态）
